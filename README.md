## ğŸ§© Puzzle Tasks
Players must complete the following tasks:

1. **Collect and drop items correctly**  
   - Match each item to its corresponding target (Naruto, Luffy, Yuji, Ichigo, Goku).  
2. **Avoid wrong drops**  
   - Dropping the wrong item or placing it in the wrong location costs a life.  
3. **Clear all targets per level**  
   - Destroy all targets to revive heroes and complete the level.
